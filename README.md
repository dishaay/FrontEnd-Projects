
A collection of my **frontend projects** built using **HTML and CSS**, all organized in one place.
This repository showcases my practice work, design experiments, and mini projects as I learn and grow in frontend development.


## What’s Inside

Each folder in this repository represents a **separate frontend project**:

* **Clara Bow Portfolio** – A themed portfolio-style webpage
* **Conan Gray** – A fan-based or artist-inspired webpage
* **Maya Hawke** – A design-focused frontend project
* **Venue Booking 1 & 2** – Simple venue booking UI layouts
* **images** – Shared assets used across projects

All projects are built using **pure HTML and CSS** (no frameworks).



## Tech Stack

* HTML5
* CSS3



## Purpose of This Repository

* To practice frontend fundamentals
* To improve UI layout and styling skills
* To maintain all frontend projects in one organized repo
* To track learning progress over time



## How to View the Projects

1. Open any project folder
2. Locate the `index.html` file
3. Open it in your browser

*No additional setup required.*
